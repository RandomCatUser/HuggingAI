# ✨ AI Local Chat App — GPT-4o Style

> 💬 Chat with your personal AI — right from your PC, Phone, or Tablet.  
> 🌟 Private. Beautiful. Fast. Built with ❤️ using Hugging Face APIs.

---

## 🎨 Features
✨ Modern GPT-4o Style UI — smooth bubbles, clean animations  
⚡ Instant Local Response — hosted on your own Wi-Fi  
📱 Multi-Device Support — PC, phone, tablet — no setup  
🔒 Fully Private — no cloud, no spying  
🛠️ Single-File App — everything packed inside one Python file  
🧩 Easy to Customize — themes, models, animations

---

## 📋 System Requirements
- 🐍 Python **3.8+**
- 📦 Packages: `flask`, `requests`
- 🌐 Hugging Face API Key (Free or Paid)

> _Tip:_  
> Get your Hugging Face API key from 👉 [https://huggingface.co/settings/tokens](https://huggingface.co/settings/tokens)

---

## 🚀 Installation

```bash
# 1. Download the app
git clone https://github.com/RandomCatUser/HuggingAI.git
cd HuggingAI

# 2. Install dependencies
pip install flask requests

# 3. Set your Hugging Face API Key inside app.py

# 4. Run the app
python app.py
